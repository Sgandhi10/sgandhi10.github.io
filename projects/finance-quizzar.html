Finance QuizzAR

Developer | Interactive Education
Unity C# Arduino Python

Project Overview

Finance QuizzAR is a 3D educational game developed in Unity designed to impart fundamental finance concepts to younger audiences. By blending digital gaming with physical hardware interactions, the project aimed to make learning financial literacy highly engaging.

The Challenge

The primary technical hurdle was bridging the gap between a custom physical controller (powered by an Arduino) and the 3D Unity game engine. Native serial communication between Unity and external microcontrollers can be highly unstable, leading to input lag or crashed environments.

The Solution

Instead of forcing a direct connection, I utilized Python as a middleware layer. Python safely managed the serial connection to the Arduino, processed the raw sensor data, and simulated keystrokes/interactions that the Unity engine could natively understand without instability.

Technical Implementations

  • Game Mechanics: Programmed custom physics constraints and character controllers in C# within the Unity environment to create a smooth, responsive gameplay experience.
  • Hardware Integration: Wired and programmed an Arduino to read physical inputs (buttons/sensors) and reliably transmit that data via serial communication.
  • Middleware Scripting: Authored Python scripts to act as a stable translation layer, ensuring hardware inputs translated instantly to in-game actions with zero dropped packets.
