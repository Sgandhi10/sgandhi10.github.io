Band Aid Together

Developer | IoT & Cloud Architecture
IoT Google Firebase Arduino C++

Project Overview

Band Aid Together was an Internet of Things (IoT) initiative focused on securely and rapidly transmitting local sensor data to a cloud environment. I was responsible for overhauling the communication architecture of the device to ensure real-time data sync with minimal latency.

The Challenge

The initial prototype was built on an Arduino Nano BLE 33 Sense, which relied heavily on proximity-based Bluetooth connections. This severely limited the range of the device and made it impossible to remotely monitor the sensor data or sync it across multiple users globally in real-time.

The Solution

I transitioned the hardware stack to an Arduino MKR Wi-Fi 1010 and completely rebuilt the networking protocol. By connecting the device directly to local Wi-Fi, I established a secure, continuous pipeline to a Google Firebase real-time database, enabling instant global access to the hardware's data.

Technical Implementations

  • Hardware Transition: Ported existing sensor logic from the BLE 33 Sense to the MKR Wi-Fi 1010, ensuring no loss in sensor accuracy while establishing a robust internet connection.
  • Firebase Integration: Utilized the Firebase REST API to structure and push JSON data payloads directly from the C++ microcontroller code to the cloud without intermediate server processing.
  • Latency Optimization: Optimized the data polling rates to reduce battery consumption while maintaining a real-time feel on the frontend dashboard reading the Firebase data.
