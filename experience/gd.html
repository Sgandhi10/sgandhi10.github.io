Software Engineering Intern | Fairfax, VA | May 2022 - Aug 2022
Linux / Bash Python NFS / DHCP / TFTP Cybersecurity (DoD STIG)

Overview

During my time at General Dynamics Mission Systems, my work focused on bridging software engineering with IT infrastructure and cybersecurity. I was responsible for modernizing and securing the operating system deployment pipeline for mission-critical hardware, ensuring that systems could be spun up rapidly without compromising strict defense-grade security protocols.

Key Impact & Responsibilities

  • Automated Provisioning: Engineered an automated Linux OS deployment pipeline over the network (NFS) utilizing DHCP and TFTP. This eliminated manual OS installation bottlenecks and cut overall deployment time by 50%.
  • Security Hardening: Defense systems require rigorous compliance. I developed custom Bash and Python scripts to harden system kickstarts, ensuring all newly deployed operating systems automatically complied with Department of Defense (DoD) Security Technical Implementation Guide (STIG) requirements right out of the box.
  • Infrastructure Efficiency: By automating both the deployment and the security hardening phases, I significantly reduced the manual administrative overhead required by the engineering team to prep hardware for deployment.
