Project Overview

At the Building Efficient and Resilient Hardware (BERH2) Lab, I researched and developed advanced hardware security measures. I focused on designing custom hardware-level encryption instructions to mitigate modern microarchitectural side-channel attacks.

View Paper (arXiv)

The Challenge

Modern processors are vulnerable to speculative execution vulnerabilities like Spectre. Defending against these requires introducing overhead or specialized logic. The challenge was designing a robust protection mechanism that leverages cryptographic concepts without degrading overall system performance.

The Solution

I developed a custom pointer encryption assembly instruction inspired by ARM's Pointer Authentication Code (PAC). Using SystemVerilog, I integrated Prince Encryption alongside the BOOM core architecture, ensuring hardware-level resilience against Spectre attacks while validating the modified processor design.
