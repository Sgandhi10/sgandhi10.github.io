Politilink

HackViolet 2023 | Best Innovation Hack
React Flask GPT-3 NLP Web Scraping

Project Overview

Politilink is an automated civic engagement platform designed to lower the barrier to entry for political activism. The application uses natural language processing and web scraping to empower users to effortlessly contact their correct local representatives regarding the issues they care about most.

The Challenge

Civic engagement often suffers because writing customized, well-researched emails to local representatives is time-consuming and confusing. Constituents frequently do not know who their specific local representatives are based on their exact district, nor do they know how to properly format an advocacy letter.

The Solution

I developed a web application that takes a user's ZIP code and a brief summary of their stance on an issue. The backend scrapes public databases to identify the correct representative, and then leverages the OpenAI GPT-3 API to draft a highly researched, professional, and persuasive email tailored to that specific politician.

Technical Implementations

  • AI Integration: Engineered specific prompt pipelines utilizing the OpenAI GPT-3 API to ensure the generated letters maintained a professional tone while accurately reflecting the user's specific policy preferences.
  • Data Scraping & Routing: Implemented web scraping scripts in Python to map user ZIP codes directly to the correct state and local representatives, bridging the gap between national APIs and localized data.
  • Full-Stack Execution: Built a fluid, accessible frontend using React and connected it to a Flask backend, allowing for rapid API calls and seamless user experiences.
