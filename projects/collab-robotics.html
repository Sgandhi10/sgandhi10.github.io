Collaborative Robotics Lab

Undergraduate Researcher | 3rd Place at CHCI 2023
ROS (Robot Operating System) React Human-Robot Interaction

Project Overview

Working as an Undergraduate Researcher, I spearheaded the front-end software development to interface with a Fetch Mobile Manipulator robot. The goal was to build a system to study and improve implicit communication and trust between humans and collaborative robots in shared workspaces.

The Challenge

Industrial and service robots often lack intuitive ways to communicate their internal states or intended actions to human partners, leading to inefficiency and lack of trust. We needed a robust interface to connect the robot's complex backend processing with a comprehensible user interface during active user studies.

The Solution

I developed a responsive web application using React and ROSLibJS. This architecture allowed the React frontend to subscribe to and publish ROS topics in real-time over WebSockets, establishing a seamless two-way communication channel between the human operator and the Fetch Robot.

Research & Impact

  • User Studies: Facilitated comprehensive user studies to map how varying speeds, proximity, and visual feedback from the robot affected human trust metrics.
  • System Architecture: Successfully bridged standard web development stacks (React) with complex robotic middleware (ROS), allowing for rapid iteration of the user interface without altering the core robotic navigation stack.
  • Recognition: Our team's research and resulting platform were awarded 3rd Place at the Center for Human-Computer Interaction (CHCI) showcase in 2023.
