Project Overview

Latis is a cybersecurity initiative designed to track and manage security updates for medical devices. The goal of the startup was to create a robust, scalable platform capable of handling sensitive security data while providing a seamless interface for administrators and clients to monitor device vulnerabilities.

The Challenge

Medical devices often rely on outdated software, making them prime targets for cyberattacks. The challenge was building an architecture secure enough to handle healthcare-adjacent tech data, while also being agile enough to scale from a prototype to a market-ready product quickly.

The Solution

I developed a full-stack architecture using Next.js and VueJS for a dynamic, responsive front-end, paired with a Python/Flask and PostgreSQL backend for secure data management. To ensure consistent deployment environments, I fully containerized the application using Docker.
